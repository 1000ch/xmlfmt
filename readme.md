# xmlfmt [![Build Status](https://travis-ci.org/1000ch/xmlfmt.svg?branch=master)](https://travis-ci.org/1000ch/xmlfmt)

Format XML string.

## Usage

```javascript
const xmlfmt = require('xmlfmt');

xmlfmt('<foo>Foo!</foo>');
```
