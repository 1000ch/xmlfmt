# xmlfmt ![test](https://github.com/1000ch/xmlfmt/actions/workflows/test.yml/badge.svg)

Format XML string.

## Install

```sh
npm install xmlfmt
```

## Usage

```javascript
const xmlfmt = require('xmlfmt');

xmlfmt('<foo>Foo!</foo>');
```
